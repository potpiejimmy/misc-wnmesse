This folder contains:

This is the project's root folder that contains the client, server and common
root folders.

server - contains all the server application files, such as enterprise and web
         applications, database model and scripts, server tools etc.

client - contains client application files, such as Android or iOS native apps,
         standalone clients etc.

common - contains common files that are not related to specific client or
         server application versions, such as general-purpose documentation
         files, media files, logo templates etc.
